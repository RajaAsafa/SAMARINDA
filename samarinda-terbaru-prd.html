Deskripsi Produk

Samarinda Terbaru adalah web portal berita modern yang berfungsi sebagai sumber informasi terkini, khususnya untuk wilayah Samarinda, Kalimantan Timur. Website dirancang dengan tampilan profesional, responsif, dan mudah digunakan oleh pengguna umum maupun administrator konten.

🎯 Tujuan Utama: Menjadi portal berita digital terpercaya untuk masyarakat Samarinda dengan pengalaman membaca yang optimal di semua perangkat.
  • Menyajikan berita terkini dari Samarinda secara real-time
  • Pengalaman membaca optimal di perangkat mobile maupun desktop
  • Panel admin lengkap untuk pengelolaan konten secara efisien
  • Pengelolaan siklus hidup berita otomatis dengan fitur auto-expired

07

Logika Kadaluarsa Berita

1
Berita Dibuat

Backend secara otomatis mengisi field expired_at saat berita pertama kali disimpan.

expired_at = created_at + INTERVAL '30 days'
2
Query Filter Aktif

Setiap pengambilan berita untuk publik menyertakan filter waktu kadaluarsa.

WHERE expired_at > NOW()
3
Data Retention

Berita kadaluarsa tidak ditampilkan ke publik, namun data tetap tersimpan di database. Admin dapat mengakses, mengelola, dan memperpanjang masa aktif berita kapan saja.

⚠️ Catatan: Berita kadaluarsa bersifat soft-hidden — data tidak dihapus dari database, hanya disembunyikan dari tampilan publik.

Catatan Tambahan

🧹Gunakan prinsip clean code: penamaan variabel deskriptif, fungsi singkat & terfokus.
📂Terapkan struktur folder rapi dan konsisten antara frontend dan backend.
🔄Gunakan Context API (atau Redux jika diperlukan) untuk state management di frontend.
🛡️Implementasikan JWT middleware untuk melindungi semua endpoint admin di backend.
✅Semua input dari pengguna wajib divalidasi di sisi server (server-side validation).
🔒Password admin harus di-hash menggunakan bcrypt sebelum disimpan ke database.
🔑Gunakan environment variables (.env) untuk menyimpan konfigurasi sensitif.
⚠️Terapkan error handling yang konsisten dan informatif di seluruh aplikasi.
